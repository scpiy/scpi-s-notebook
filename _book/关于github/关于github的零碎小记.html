关于github的零碎小记

如果你换了一个电脑，想要clone自己的远程仓库，请务必使用Clone with SSH而不要使用Clone with HTTPS，前者可以避免你每次push或pull的时候输入账号密码，后者虽然也能通过本地存储账号密码实现避免输入账号密码，但果然还是没有前者舒服。
